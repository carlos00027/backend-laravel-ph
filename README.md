## Instalacion de dependencias por composer

```
composer install 
```
## Ajustar variables de entorno

duplicar el archivo .env.example por .env setear credenciales de acceso a la base de datos

```
php artisan migrate
```
## Ejecutar servidor de pruebas de artisan

```
php artisan serve
```
